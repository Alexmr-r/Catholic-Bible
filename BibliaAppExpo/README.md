## 🚀 INICIO RÁPIDO

### 1️⃣ Instala Expo Go en tu móvil
- **iOS**: App Store → "Expo Go"
- **Android**: Play Store → "Expo Go"

### 2️⃣ Ejecuta y escanea el QR
```bash
npx expo start
```

**¡Escanea el QR que aparece y listo!** 📱
