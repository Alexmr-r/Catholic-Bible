# 🌐 CatholicVerse Web — Landing pública

Web estática (HTML/CSS/JS, sin framework) desplegada en Cloudflare Pages con dominio `https://getcatholicverse.com`.

- `index.html` — landing de marketing y enlaces de descarga de la app
- `privacy.html` / `terms.html` — política de privacidad y términos (requisito de App Store / Google Play)

## Desplegar cambios

```bash
npm install      # solo la primera vez
npm run deploy   # publica a Cloudflare Pages
```

Más detalle: [`../docs/05-despliegue/GUIA_DESPLIEGUE.md`](../docs/05-despliegue/GUIA_DESPLIEGUE.md) (sección 4).
